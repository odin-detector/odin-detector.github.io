Odin Detector
=============

Odin Detector is a software framework for the control of and acquisition of data from parallel 
detector systems.  The framework is modular and is built upon the Tornado Python web framework.

Documentation
-------------

The following documentation is available for Odin Detector:

* The :doc:`introduction <introduction>` section provides an overview of the two software repositories and a guide to the odin-detector framework.
* The :doc:`quickstart <quickstart>` guide will show you how to run up the software with very little detail on each step.  This guide focuses on getting running quickly and uses a simulator to show the software working without the need for any hardware.
* The :doc:`installation <installation>` instructions explain in detail the steps necessary to install the software modules on a native linux OS.
* The :doc:`user guide <userguide>` demonstrates how to run and interact with the parts of an Odin detector system.
* The :doc:`developer guide <developerguide>` contains guides for creating custom control adapters and data acquisition plugins.
* The :doc:`reference <reference>` documentation provides full details of the interactions between components as well as code generated documentation.
